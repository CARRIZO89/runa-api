Runa Api
==========================================

Enviroment
------------------------------------------

- Ruby version: `2.5.1p57`
- Rails version: `5.2.2`

Configuration
------------------------------------------

- `bundle install`
- `rails db:create`
- `rails db:migrate`
- `rails db:seed`

Usage
------------------------------------------

- `bundle exec rails server`
- Run the react app: `https://github.com/CARRIZO89/runa-front`

Run Test
------------------------------------------

- `bundle exec rspec`
